# Brennan Johnson — Portfolio Site

A simple 4-page portfolio site (Home, About Me, and 3 project pages) built with plain HTML, CSS, and JavaScript, ready to host on GitHub Pages.

## File structure

```
index.html                      Home page
about.html                      About Me page
projects/
  academic-coaching.html        Academic Coaching Essentials project page
  email-etiquette.html          How to Be Successful in College project page
  empower-program.html          Empower Program project page
css/style.css                   All site styles
js/script.js                    Mobile nav toggle
assets/
  branding/logo.png             Compass logo
  branding/brennan.jpg          Headshot
  academic-coaching/            Job aid images for that project
  email-etiquette/              Job aid image for that project
```

## Updating content later

- **Empower Program slides:** once the slide deck link is ready, open `projects/empower-program.html`, find the `coming-soon-card` paragraph under "Deliverables," and replace it with a linked image or button, following the same pattern used in the other two project pages.
- **Video walkthroughs:** currently linked directly to Google Drive. If you move them to YouTube (recommended for reliability), replace the `video-link-card` link with an embedded iframe.
